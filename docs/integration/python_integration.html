Python Integration

To demonstrate the SLQ method in python, we start with a simple symmetric matrix A \in \mathbb{R}^{n \times n}.

import numpy as np 
from scipy.linalg import toeplitz
X = toeplitz(np.random.uniform(size=50))
A = X.T @ X
